### **Installation Instructions**
Since this is a **Node.js project**, 
instead of `pip install -r requirements.txt`, use:

#### **For Backend**
```bash
cd server
npm install
npm start
```

#### **For Frontend**
```bash
cd client
npm install
npm run dev
```
